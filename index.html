C++

C++ is a cross-platform language that can be used to create high-performance applications.

C++ was developed by Bjarne Stroustrup, as an extension to the C language.

C++ gives programmers a high level of control over system resources and memory. C++ is an Object Oriented Programming language but is not purely Object Oriented. Its features like Friend and Virtual, violate some of the very important OOPS features, rendering this language unworthy of being called completely Object Oriented. Its a middle level language.

C++ is a middle-level language rendering it the advantage of programming low-level (drivers, kernels) and even higher-level applications (games, GUI, desktop apps etc.). The basic syntax and code structure of both C and C++ are the same.

Featuers of C++:

  • Simple

    It is a simple language in the sense that programs can be broken down into logical units and parts, has a rich libray support and a variety of data-types.

  • Machine Independent but Platform Dependent

    A C++ executable is not platform-independent (compiled programs on Linux won’t run on Windows), however they are machine independent.

  • Mid-level language

    It is a mid-level language as we can do both systems-programming (drivers, kernels, networking etc.) and build large-scale user applications (Media Players, Photoshop, Game Engines etc.)

  • Rich library support:

    Has a rich library support (Both standard ~ built-in data structures, algorithms etc.) as well 3rd party libraries (e.g. Boost libraries) for fast and rapid development

  • Speed of execution

    C++ programs excel in execution speed. Since, it is a compiled language, and also hugely procedural. Newer languages have extra in-built default features such as grabage-collection, dynamic typing etc. which slow the execution of the program overall. Since there is no additional processing overhead like this in C++, it is blazing fast

  • Pointer and direct Memory-Access

    C++ provides pointer support which aids users to directly manipulate storage address. This helps in doing low-level programming (where one might need to have explicit control on the storage of variables).

C++ fully supports object-oriented programming, including the four pillars of object-oriented development

  • Encapsulation
  • Data hiding
  • Inheritance
  • Polymorphism

Uses of C++

  • C++ is used in essentially every application domain by hundreds of thousands of programmers.
  • C++ is used extensively for writing system drivers and other applications that rely on direct hardware manipulation under real-time constraints.
  • C++ is commonly used for teaching and study, as it is clean enough to teach simple concepts successfully.
  • Anyone who has either used an Apple Macintosh or a PC running Windows has indirectly used C++ because these systems' primary user interfaces are written in C++

Basic C++ Syntax

    If we consider a C++ program, it can be described as a set of objects which communicate by invoking the methods of each other. Now let's look briefly at wha a class, object, methods, and instant variables mean.

  • Object− Objects have different states and behaviors. Example: A dog has statements-colour, gender , race and behaviors-wagging, barking and eating. An object is a single instance of a class
  • Class − A class can be described as a template / blueprint describing the behaviors / states supporting the object of its kind.
  • Methods − Methods are basically conducts. A class can have several methods in it. It's in methods where the logics are written, data is manipulated, and all actions are carried out.
  • Instance Variables − Each object has its unique set of instance variables. An object's state is created by the values assigned to these instance variables.
